## Installation

* Using CocoaPods
* You can use CocoaPods to install Fuzi by adding it to your to your Podfile:

```platform :ios, '8.0'
use_frameworks!

target 'MyApp' do
	pod 'LiKui', '~> 2.0.2'
end
Then, run the following command:

$ pod install
```
